---
title: "Create Link URL"
description: "Generate a Buybase <a href=\"/docs/buybase-link\">Link</a> URL to redirect your users to."
api: "GET /link"
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication. 
</ParamField>

### Query Parameters

<ParamField query="user_id" type="string" required>
  The unique identifier for the user.
</ParamField>

<ParamField query="developer_id" type="string" required>
  The unique identifier for the developer.
</ParamField>

<ParamField query="app_id" type="string" required>
  The unique identifier for the app.
</ParamField>

<ParamField query="platform" type="string" required>
  The platform for which the Buybase link is being generated. See supported platforms [here](/docs/supported-platforms).
</ParamField>

### Response

<ResponseField name="link" type="string">
  The generated Buybase link for the user to link their e-commerce account.
</ResponseField>

<ResponseField name="code" type="string">
  A code indicating the result of the link creation operation. Possible value: `BUYBASE_LINK`.
</ResponseField>

<ResponseExample>

```json Response
{
  "link": "https://link.buybase.ai?platform=amazon&user_id=user_1234567890abcdef&developer_id=dev_1234567890abcdef&app_id=app_1234567890abcdef",
  "code": "LINK_SUCCESS"
}
```

</ResponseExample>
