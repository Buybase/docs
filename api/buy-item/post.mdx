---
title: "Buy Item"
description: "Buy an item on behalf of your end-user."
api: "POST /buy"
---

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Body

<ParamField body="user_id" type="string" required>
  The unique identifier for the user.
</ParamField>

<ParamField body="developer_id" type="string" required>
  The unique identifier for the developer.
</ParamField>

<ParamField body="app_id" type="string" required>
  The unique identifier for the app.
</ParamField>

<ParamField body="quantity" type="integer" required>
  The number of items to purchase. Must be a positive integer.
</ParamField>

<ParamField body="product_url" type="string" required>
  The URL of the product to purchase. Must be a valid URL product page URL from one of our <a href="/docs/supported-platforms">supported platforms</a>.

Example Amazon Product URL:

```javascript
"https://www.amazon.com/example-product-title/dp/B0D1XD1ZV3"
```

</ParamField>

### Response

<ResponseField name="message" type="string">
  A message indicating the result of the purchase operation.
</ResponseField>

<ResponseField name="code" type="string">
  A code indicating the result of the purchase operation.
</ResponseField>

<ResponseExample>

```json Response
{
  "message": "Purchase successful",
  "code": "PURCHASE_SUCCESS"
}
```

</ResponseExample>
