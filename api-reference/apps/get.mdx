## Introduction

Get app details by ID. This endpoint returns information about your Buybase app.

### Headers

<ParamField header="authorization" type="string" required>
  Your API key for authentication.
</ParamField>

### Query Parameters

<ParamField query="app_id" type="string" required>
  The unique identifier for the app.
</ParamField>

<ParamField query="developer_id" type="string" required>
  The unique identifier for the developer.
</ParamField>

### Response

<ResponseField name="app" type="object">
  The app details.
  1. `id` - The unique identifier for the app.
  2. `created_at` - The date and time the app was created.
  3. `name` - The name of the app.
  4. `developer_id` - The unique identifier for the developer who owns this app.
  5. `logo_url` - URL of the app's logo image, or null if no logo is set.
</ResponseField>

<ResponseField name="code" type="string">
  A code indicating the response status or type.
</ResponseField>
